# Руководство по админ-панели

## Доступ к админ-панели

### Учетные данные администратора по умолчанию:
- **Логин:** `admin`
- **Пароль:** `admin123`

После входа в систему с учетной записью администратора в навигационном меню появится кнопка "Админ-панель".

## Возможности админ-панели

### 1. Панель управления (Dashboard)
- Просмотр общей статистики сайта
- Количество книг, пользователей, комментариев и цитат
- Быстрый доступ к основным разделам

### 2. Управление книгами
**Доступные действия:**
- ✅ Добавление новых книг
- ✅ Редактирование информации о книгах (название, автор, жанр, описание, обложка, PDF)
- ✅ Удаление книг

**Обязательные поля:**
- Название книги
- Автор

**Необязательные поля:**
- Жанр
- Описание
- URL обложки
- URL PDF файла

### 3. Управление пользователями
**Доступные действия:**
- ✅ Просмотр всех пользователей
- ✅ Добавление новых пользователей
- ✅ Редактирование данных пользователей
- ✅ Изменение роли пользователя (администратор/пользователь)
- ✅ Удаление пользователей

**Важно:**
- Нельзя удалить последнего администратора
- При редактировании пользователя пароль можно оставить пустым (он не изменится)
- Email и логин должны быть уникальными

### 4. Управление контентом
Возможность редактирования всех текстовых блоков главной страницы:

**Hero секция:**
- Главный заголовок
- Подзаголовок
- Описание
- Фоновое изображение

**Секция особенностей (Features):**
- 3 карточки с заголовками, описаниями и иконками
- Поддерживаемые иконки: BookOpen, Users, MessageCircle

**Галерея:**
- 2 элемента с изображениями, заголовками и описаниями

**Секция книг:**
- Заголовок
- Описание

**Футер:**
- Текст в подвале сайта

## Хранение данных

Все данные сохраняются в localStorage браузера:
- `users` - пользователи
- `books` - книги
- `comments` - комментарии
- `quotes` - цитаты
- `siteContent` - контент главной страницы
- `currentUser` - текущий авторизованный пользователь

## Навигация

Боковое меню админ-панели включает:
- Панель управления
- Управление книгами
- Управление пользователями
- Управление контентом
- Кнопка "На главную" (возврат на сайт)
- Кнопка "Выйти"

## Уведомления

Все действия в админ-панели сопровождаются всплывающими уведомлениями:
- ✅ Успешное выполнение операции
- ❌ Ошибки валидации или выполнения

## Безопасность

**Важно:** Это демонстрационная версия. В реальном приложении необходимо:
- Использовать серверную авторизацию
- Шифровать пароли
- Использовать токены доступа
- Ограничить доступ к API на уровне сервера
